    Django==1.7.1 #(likely 1.8)

    django-durationfield==0.5.1
    django-suit==0.2.11
    django-suit-redactor==0.0.2

    South==1.0.1

Установка:
=========
    pip install -r requirements.txt
    manage.py syncdb
    manage.py runserver
